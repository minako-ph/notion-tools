# Notion API x Trello API

## 概要

TrelloをNotionに同期するスクリプト

Trello上の新規作成・更新・削除が
Notionのデータベースに同期される

Trelloにカレンダー形式でリストを作成している自分のボードに合わせて作成しています
- [Trelloカレンダーの参考](https://doctype.jp/blog/backend/507/)
